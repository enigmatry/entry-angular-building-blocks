Apply server & client side validation errors to reactive form using entry-form-errors component and entryDisplayControlValidation directive.

Use ENTRY_VALIDATION_CONFIG provider to configure client side validators messages.


Apply server & client side validation errors to Formly form using entry-form-errors component and FormlyModule configuration. Same principle applies to the forms generated with Entry Code Generator, since they are based on Formly.

Use FormlyModule to configure how field level validation messages (client & server side) will be mapped. To map server side field validation messages we use FORM_FIELD_ERROR_KEY key.


Validation errors can be mapped to complex form structures like groups and arrays.
